# George-Topuria-Library-System

## Project Overview

This project is a basic Library Management System that is created in Python.

Library staff can use the program to handle books, members and borrowing records using a menu-driven interface.

The system illustrates the use of Object Oriented Programming, File Handling, Exception Handling, Modularization and Control Structures.

## Features

* Add books
* Add members
* Borrow books
* Return books
* View books
* View members
* View loan records
Write data to CSV files.
When the program is started, load data from CSV files.
* Do not allow duplicate book IDs
- Do not allow duplicate member IDs

## Files

* main.py – main menu and user interaction
The library logic is contained in the library.py file.The logic for the library is in library.py.
* book.py – Book class
* member.py – Member class
* loan.py – Loan class
This is a file that contains book data, stored in the form of a CSV file.
This is a file containing the data of the members.This is a file that stores member data.
* loans.csv – stored loan data

## Installation

1. Install Python 3.
2. Download the project files.
3. Open the project folder in VS Code.

## Running the Program

Run the following command:

python main.py

## Example Usage

1. Add a book.

4. Return a book.
